Learn Computing

I chose the API tutorial. APIs, or Application Program Interfaces, deliver data between clients and servers. Individual users who use apps or google things put in requests called "gets" on some type of graphical user interface or GUI. Then, the app or the site asks the server for information and then delivers it back to them through "posts". Lots of businesses make free, public APIs to encourage engagement from 3rd parties. People can access the endpoints, or a URL where you can find it. The presentation mentioned Spotify's endpoints, which reminded me of this app my friends and I use called airbuds which lets you see what music your friends are listening to in real time. It must use Spotify and Apple Music API endpoints.
